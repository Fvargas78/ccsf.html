How To Create a Website

What is HTML used for?

HTML is used to create electronic documents (called pages) that are displayed on the World Wide Web. Each page contains a series of connections to other pages called hyperlinks. HTML will be used to insert the content – the many words, sentences, paragraphs – that will constitute your web pages that together will make your website.

SIMPLE STEPS TO CREATING A WEBSITE:

  • Choose a domain name
  • Pick a hosting plan
  • Choose a Platform
  • Install Wordpress
  • Pick your Theme/ Template
  • Install Plugins
  • Upload your content
